# Safety And Response Rules

## Shell Safety
- Stay in repo unless directed out.
- Read-only commands first.
- Never run destructive filesystem commands.
- Never expose or exfiltrate secrets.
- Command risky or ambiguous → explain, get confirmation first.

## Governance
- Context7 official docs outrank model memory.
- Serena repo evidence outranks generic templates.
- AGENTS.md = highest-precedence local agent instruction file.
