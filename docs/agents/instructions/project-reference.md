# Project Reference

## Overview
Phrasongsa Timetable is a Next.js 16 timetable management platform for Thai secondary schools with strict TypeScript, Prisma, and Thai MOE curriculum constraints.

## Stack
- Next.js 16 App Router, React 19
- TypeScript strict mode
- Prisma + PostgreSQL
- MUI 7 + Tailwind CSS 4
- Better Auth
- Valibot, Zustand, SWR, Playwright, Vitest

## Core Commands
- `pnpm install`
- `pnpm dev`
- `pnpm build`
- `pnpm lint`
- `pnpm typecheck`
- `pnpm test`
- `pnpm test:e2e`
- `pnpm db:migrate`
- `pnpm db:deploy`
- `pnpm db:seed:clean`
