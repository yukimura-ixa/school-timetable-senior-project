# Feature Map — Top 5 User Journeys

> Generated by `/investigate` on 2026-05-26. Each journey lists the full call path from UI to DB and back, with file paths verified during the deep pass.

Routes are protected unless marked **(public)**. Auth gate enforced in the segment `layout.tsx` (no middleware).
